submitForm

function
 

Called when the form is submitted. The object is updated by calling it's save method. The event submit is dispatched.

function(vm, form, event)

Parameters

Name Type Description
vm {Object}

The scope of the form (this view model)
form {Form}

the dom form
event {Event}

the dom form event

Returns


{Boolean}

returns false to prevent form submissions
