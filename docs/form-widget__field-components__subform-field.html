<subform-field />

module
{can.Component}
form-widget/field-components/subform-field  

This component creates a sub form fieldset that is sub object when values in the sub form change. This can be used to store nested data in a database field or group related fields. This is a replacement for the json-field to enable the ability to work with objects as opposed to json strings.

Example

{
  ObjectTemplate: DefineMap.extend({
    //object properties
    field1: 'string',
    field2: 'number'
  }),
  fieldType: 'subform'
}
