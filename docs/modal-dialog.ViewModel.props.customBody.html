customBody

property
{HTMLBoolean}
modal-dialog.ViewModel.props.customBody  

Whether or not to use a custom body content for the modal. The default is false. This overrides the title property and modal-body divs displayed by displaying only the content passed to the <modal-dialog></modal-dialog> component
